HTML : Structure, Balises, Liens et Navigation

🎯 Introduction au HTML

HTML (HyperText Markup Language) est le langage de balisage utilisé pour créer des pages web. Ce n'est pas un langage de programmation, mais un langage de structuration qui permet d'organiser le contenu.

À retenir : HTML donne la structure (les murs et les pièces), CSS viendra pour le style (la décoration), et JavaScript pour l'interactivité (l'électricité).

🛠️ Installer les outils nécessaires

1. Éditeur de texte

Outil pour écrire le code : Visual Studio Code (recommandé, gratuit), Sublime Text, ou Notepad++.

2. Navigateur web

Google Chrome, Firefox, Edge - pour afficher et tester vos pages. Utilisez F12 pour voir les outils développeur.

🏗️ Structure fondamentale d'une page HTML

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon premier site</title>
</head>
<body>
    <!-- Tout le contenu visible ici -->
</body>
</html>
  • <!DOCTYPE html> : Indique au navigateur qu'il s'agit de HTML5
  • <html lang="fr"> : Racine du document, langue française
  • <head> : Métadonnées (non visible)
  • <meta charset="UTF-8"> : Gère les accents (é, è, ç, etc.)
  • <title> : Titre dans l'onglet du navigateur
  • <body> : Corps de la page, tout ce qui s'affiche

📌 Les titres : <h1> à <h6>

Hiérarchie du contenu :

<h1>Titre principal (1 seul par page)</h1>
<h2>Section importante</h2>
<h3>Sous-section</h3>
<h4>Détail</h4>
<h5>Sous-détail</h5>
<h6>Très petit détail</h6>

📝 Paragraphes et sauts de ligne

<p>Ceci est un paragraphe. Il contient plusieurs phrases.</p>
<p>Deuxième paragraphe avec <br> un saut de ligne<br>et une autre ligne.</p>

La balise <br> est auto-fermante et permet un simple retour à la ligne.

✨ Mettre en valeur : <strong> et <em>

<p>Ceci est <strong>très important</strong> et ceci est <em>mis en emphase</em>.</p>
  • <strong> : gras, importance forte
  • <em> : italique, emphase légère

📋 Les listes

Liste non ordonnée (à puces)

<ul>
    <li>Pomme</li>
    <li>Banane</li>
    <li>Cerise</li>
</ul>

Liste ordonnée (numérotée)

<ol>
    <li>Préchauffer le four</li>
    <li>Mélanger les ingrédients</li>
    <li>Enfourner</li>
</ol>

🔗 Les liens hypertextes <a>

Syntaxe de base : <a href="cible">Texte cliquable</a>

Liens dans le même dossier

<a href="contact.html">Contact</a>

Liens dans un sous-dossier

<a href="pages/apropos.html">À propos</a>

Liens vers dossier parent

<a href="../index.html">Accueil</a>

Liens externes (nouvel onglet)

<a href="https://google.com" target="_blank">Google</a>

Liens internes (ancres)

<h2 id="top">Titre</h2>
<a href="#top">Retour en haut</a>

Email et téléphone

<a href="mailto:contact@email.com">Envoyer mail</a>
<a href="tel:+243123456789">Appeler</a>

✅ Bonnes pratiques

  • Indenter son code pour la lisibilité
  • Toujours fermer les balises
  • Noms de fichiers : minuscules, sans accents, tirets à la place des espaces
  • Tester régulièrement dans le navigateur (F5 pour actualiser)
  • Utiliser des commentaires <!-- ceci est un commentaire -->
Résumé : Avec ces balises, vous pouvez créer des pages structurées, des listes, et des liens entre plusieurs pages !

TP N°1 : Mini Site Web 3 Pages

🧪 Création d'un site web complet

Mettez en pratique toutes les balises HTML vues en cours !

🎯 Objectif

Réaliser un mini site web composé de 3 pages reliées entre elles, en utilisant uniquement le HTML et les balises vues en cours.

📁 Organisation du travail

Créez un dossier nommé mon-ecole. À l'intérieur, créez :

mon-ecole/
│
├── index.html      (Page d'accueil)
├── apropos.html    (Page À propos)
└── contact.html    (Page Contact)

📝 Consignes générales

  • Chaque page doit avoir la structure complète : <!DOCTYPE html>, <html lang="fr">, <head>, <meta charset="UTF-8">, <title>, <body>
  • Bien indenter le code (propre et lisible)
  • Fermer correctement toutes les balises
  • Tester chaque page dans le navigateur

🏠 PAGE 1 : ACCUEIL (index.html)

  • Un titre principal <h1>
  • Au moins 2 paragraphes <p>
  • Un saut de ligne <br>
  • Utilisation de <strong> et <em>
  • Un sous-titre <h2>
  • Une liste non ordonnée <ul>
  • Une liste ordonnée <ol>
  • Une section Navigation avec liens vers apropos.html et contact.html

📖 PAGE 2 : À PROPOS (apropos.html)

  • Un titre <h1>
  • Au moins 2 paragraphes
  • Un sous-titre <h2> avec liste non ordonnée
  • Un autre sous-titre <h2> avec liste ordonnée
  • Navigation vers index.html et contact.html

📞 PAGE 3 : CONTACT (contact.html)

  • Un titre <h1>
  • Un paragraphe de bienvenue
  • Un sous-titre <h2> avec liste non ordonnée (coordonnées)
  • Une liste ordonnée (ex: horaires)
  • Utilisation de <strong>
  • Navigation vers index.html et apropos.html

🔗 Navigation obligatoire

Toutes les pages doivent être reliées entre elles. Aucun lien ne doit être cassé.

⭐ Bonus (facultatif)

  • Ajouter un lien externe https://... avec target="_blank"
  • Ajouter un commentaire HTML <!-- -->
  • Créer une ancre pour retour en haut de page
Rendre votre travail : Une fois les 3 pages créées et testées, compressez votre dossier mon-ecole OU uploadez vos fichiers ci-dessous.
